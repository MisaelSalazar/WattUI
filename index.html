Instalación

Mobile‑first

Copia el CSS embebido de este archivo a ./css/wattui.css y enlázalo en tus vistas:

<link rel="stylesheet" href="/css/wattui.css" />

Recomendación de colores: --watt-orange y superficies crema para emular la vibra Wattpad.
